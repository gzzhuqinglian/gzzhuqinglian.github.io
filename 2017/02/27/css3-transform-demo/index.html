前言

随着web的发展，对界面的视觉要求越来越高，在页面中增加动画，是一种特别好的一件事情，但是我对动画属性掌握的不是那么好，特别的抽时间去学习和总结。

1.css的动画基本属性的学习
2.css的动画的小demo的展现
3.市场上现有的动画的框架的分析

css3 transform 变形
首先由2d变形的相关api。translate位置变化，scale大小变化，rotate角度变化，skew是dom本身的形状的变化。
其次是3d变化的相关api。在2d的基础上增加了3维的变化，但是，3维的变化针对的dom本身的变化，通俗来讲，当一个dom的绕着y坐标旋转以后，在进行z坐标的变化是不一样的。

css3 transition 过渡。
表示一个属性从从一个状态到其他的状态的变化过程。
一般来讲，，变化过程，有这么些属性控制，，过渡的属性，过渡的时间，过渡的函数，过渡的延时。
需要注意一点的是过渡的函数，不仅仅是常用的几种过渡函数，还有cubic-bezier表示，具体的可以点http://cubic-bezier.com/#.17,.67,.83,.67查看
需要声明一点过渡是由于状态的变化造成的，所以是事件触发。一般是一次性的。

css3 animation 动画

说起动画，动画是一组过程组成。也就是我们常说的帧，属性keyframes 。
动画同时也有属性去限制他的相关的一些动画过程。
动画时间，动画函数，动画延时，动画次数，动画方向。
可以参考w3c的基本demo
http://www.w3school.com.cn/css3/css3_animation.asp
